## Module Aim

This module focuses on the Web Design and Development Process, from User Experience (UX), Planning, Responsive Design, to Development and Maintenance. The module will use industry standard website planning, design and web authoring tools. It will include web publishing and cross platform application development.

---

## Learning Outcomes

On successful completion of this module, learners will be able to:

1. Demonstrate an ability to document and complete the planning stage of a Web Development project.
2. Apply Web UX guidelines to the design of an interactive website.
3. Demonstrate an ability to construct a responsive website using a Content Management System.
4. Demonstrate an ability to develop an interactive website using industry standard web authoring tools.
5. Publish and test an interactive website.

---

## Indicative Content

The module covers the following topics:

- Planning & User Experience
- Design & User Experience
- Development: Content Management Systems, What You See Is What You Get (WYSIWYG) Authoring
- Web Publishing
- Responsive Development / Frameworks (Cross Platform App Development)
- Website Maintenance: Search Engine Optimisation , Testing & Approval, Launch & Transition

---

## Learning and Teaching Methods

This is a lab based module. This module will be delivered using 2 hours of computer-based Lectures, along with 2 hour of computer-based Practicals each week.

The Lectures will introduce new concepts and theory relating to the web development process. The practicals will allow the students to design and develop websites using industry standard web authoring software.

## Assessment Criteria

### Fail (<40%)
<40%: Unable to interpret and describe key concepts of the web development process.

### Pass (40%-49%)
40%-49%: Be able to interpret and describe key concepts of the web development process.

### Credit (50%-59%)
50%-59%: Ability to discuss key concepts of the web development process and ability to discover and integrate related knowledge in other knowledge domains.

### Distinction (60%-69%)
60%-69%: Be able to solve problems within web development by experimenting with the appropriate skills and tools.

### High Distinction (70%-100%)
70%-100%: All the above to an excellent level. Be able to analyse and design solutions to a high standard for a range of both complex and unforeseen problems through the use and modification of appropriate skills and tools.
