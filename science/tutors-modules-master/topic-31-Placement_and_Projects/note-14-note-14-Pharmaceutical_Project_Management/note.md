## Module Aim

This module aims to provide the learner with the ability to develop an understanding of project management and its application. It aims to demonstrate the importance of the planning process in ensuring project success, to provide the student with the ability to define and build project plans, to estimate project durations using appropriate software tools, and to apply appropriate tools and techniques to control and monitor project schedules.

---

## Learning Outcomes

On successful completion of this module, learners will be able to:

1. Evaluate the application of the principles of project management.
2. Design, build and troubleshoot project management plans.
3. Estimate project durations and Create, control, monitor project schedule using appropriate software tools (Software and non software).
4. Estimate project costs, report on earned value, analyse cost variances and produce estimates at completion.
5. Evaluate the project group aspects of project management - organisation, roles, attributes of the project manager, the role of sponsor etc.

---

## Indicative Content

The module covers the following topics:

- Introduction to project management
- Project life cycle
- Project design and planning: work breakdown structure, critical path network, project risk management, project change management, project quality control
- Project execution: reporting, documentations required and communication etc.
- Project cost measurement and management: the concept of earned value
- Managing a project team: leadership, resource allocation, HR implications, the role of the project manager
- Using project management tools and techniques
