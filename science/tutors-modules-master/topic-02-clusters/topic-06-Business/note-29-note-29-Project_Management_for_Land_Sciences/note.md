## Module Aim

This module aims to provide the student with the ability to develop an understanding of project management in context, to show the importance of the planning process in ensuring project success, to provide the student with the ability to define and build project plans, to estimate project durations using appropriate software tools, and to apply appropriate tools and techniques to control and monitor project schedules.

---

## Learning Outcomes

On successful completion of this module, learners will be able to:

1. Evaluate the application of the principles of project management.
2. Design and build project management plans.
3. Estimate project durations using appropriate software tools.
4. Create, control, monitor project schedules using project management software.
5. Evaluate the HR aspects of project management - organisation, roles, attributes of the project manager, the role of sponsor etc.
6. Estimate project costs, produce project cash flow statements, report on earned value, analyse cost variances and produce estimates at completion.
7. Evaluate the design and management plans for a variety of projects.

---

## Indicative Content

The module covers the following topics:

- Introduction to project management
- Project life cycle
- Feasibility study
- Project design and planning: work breakdown structure, critical path network, project risk management, project change management, project quality control
- Project execution: reporting, communication etc.
- Project completion and handover
- Project cost measurement and management: the concept of earned value
- Project documentation
- Managing a project team: leadership, resource allocation, HR implications, the role of the project manager
- Using project management tools and techniques
