## Module Aim

This module will enable students to produce simple landscape design plans using computer aided design (CAD), and to print out drawings to scale.

---

## Learning Outcomes

On successful completion of this module, learners will be able to:

1. Operate CAD software drawing and modifying tools.
2. Complete simple landscape drawing using CAD.
3. Create drawings and plot them.
4. Produce printed drawings to scale.

---

## Indicative Content

The module covers the following topics:

- Identify components of a CAD system
- Use quick setup wizard
- Use menu commands
- Use snap, grid, ortho, object snap
- Use scale grids
- Use drawing tools
- Use layers
- Dimensions and leaders
- Produce a hard copy of a file
- Plot to scale
