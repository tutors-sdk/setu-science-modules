# AI for Games

This module provides students with theoretical and practical concepts for the implementation and development of artificial intelligence in video games.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Game Development

---

## Learning Outcomes

- Develop artificial intelligence for a 2D or 3D game using C# (or equivalent) and demonstrate how it can improve the game play.
- Design and implement path finding and obstacle avoidance techniques.
- Construct and implement a finite state machine.
- Design and implement behaviour trees and machine learning.
- Implement and assess common AI principles and algorithms for video games.

---

## Indicative Content

- Static and dynamic path finding
- Behaviour trees
- Obstacle avoidance
- Machine learning
- Finite state machines
