# Cloud Application Services

The focus of this module is on the principles and practices in the design and implementation of Cloud based applications and supporting services.

---

## Module Details

- **Credits:** 10
- **Level:** Postgraduate
- **Department:** Computing and Mathematics
- **Cluster:** Networks and Cloud

---

## Learning Outcomes

- Assess the properties of Cloud-based Application environments (e.g. scalability, loose coupling etc.) and recognise the advantages and challenges in creating such Systems.
- Model cloud-based services that are secure, cohesive, and loosely coupled.
- Compare and contrast common Application Architectures.
- Develop and configure solutions for modern Cloud Application systems.
- Critique key features and design decisions that affect a design solution.

---

## Indicative Content

- Cloud Computing Architectures and Services
- Networking services: IP Addressing, DNS, DHCP, NAT
- Security services for Cloud Applications
- Highly Available Applications
- Web Services
- Software Defined Services
