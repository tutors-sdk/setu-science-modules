# Cloud Computing 1

This module explores key technologies in the provision of Cloud computing services.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Networks and Cloud

---

## Learning Outcomes

- Compare and contrast Cloud Computing Architectures.
- Develop appropriate Data Center Technologies in the provision of Virtualised environments.
- Evaluate Cloud technologies for the provision of Infrastructure and Application Services.
- Determine the technical challenges and solutions in providing a secure cloud services environment.
- Compare solutions for the provision of highly available, reliable, scalable Cloud environments.

---

## Indicative Content

- Cloud Computing Concepts: IaaS; PaaS; SaaS; `X'aaS
- Data Center Management: Hypervisors; Compute; Storage; Network
- Reliability and Elasticity: Infrastructure & Applications
- Monitoring: Infrastructure & Applications
- Business Continuity & Recovery
- Cloud Security: Identity and Access Management
