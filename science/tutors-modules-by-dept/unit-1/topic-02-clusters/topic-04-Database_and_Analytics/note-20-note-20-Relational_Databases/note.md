# Relational Databases

This module will introduce the student to the principles and practice of designing and implementing relational database systems.

---

## Module Details

- **Credits:** 5
- **Level:** Introductory
- **Department:** Computing and Mathematics
- **Cluster:** Database and Analytics

---

## Learning Outcomes

- Examine a business scenario to design a suitable relational database solution.
- Construct entity relationship (ER) diagrams from business scenarios and reproduce those diagrams as normalised relations ready for database implementation.
- Construct a physical database design.
- Create a relational database in business using SQL Data Definition Language (DDL).
- Construct queries on a relational database using SQL Data Manipulation Language (DML).
- Discuss the concepts of data persistence, ACID transaction management and database distribution.

---

## Indicative Content

- The Relational Model and Relational Database Management System (RDBMS)
- Database Analysis & Design
- Entity Relationship Modelling & Normalisation
- SQL Data definition & manipulation
- Data persistence, ACID transaction management and distributed databases
