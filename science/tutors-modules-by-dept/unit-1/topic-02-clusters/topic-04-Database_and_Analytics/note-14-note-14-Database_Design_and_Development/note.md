# DB Design and Development

This module will introduce the student to the principles and practice of designing and implementing efficient and secure database systems.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Database and Analytics

---

## Learning Outcomes

- Appraise the role of a database and its management system.
- Create Entity Relationship (ER) diagrams from business scenarios and reproduce those diagrams into normalised relations ready for database implementation.
- Construct a physical database design.
- Implement a relational database using SQL Data Definition Language (DDL).
- Construct queries on a relational database using SQL Data Manipulation Language (DML).
- Investigate Database Security Procedures; Introduce concepts of Data Protection (DGPR, CCPA).
- Introduce NoSQL Databases

---

## Indicative Content

- Database Environment; Relational Database Analysis and Design
- ER modelling of DB concepts
- Development of Conceptual model to Logical model
- Physical Database Design;
- Database Implementation
- Database Security & Data Protection
- Introduction to NoSQL Databases
