# Data Mining 2

It is assumed the student is familiar with the fundamental concepts and techniques of Data Mining.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Database and Analytics

---

## Learning Outcomes

- Summarise the Data Mining process and have a clear understanding of its stages.
- Evaluate the fundamental concepts behind each stage of the Data Mining process.
- Justify the use of appropriate tools and techniques for each stage of the Data Mining process.
- Evaluate, interpret and utilize results obtained at each step of the Data Mining process.
- Create a solution for a set of Data Mining problems.

---

## Indicative Content

- Introduction to the Data Mining Process
- Pre-processing: data gathering, wrangling, and transformation
- Model building, optimization and evaluation
- Result analysis, validation, deployment
- Use of data mining tools
