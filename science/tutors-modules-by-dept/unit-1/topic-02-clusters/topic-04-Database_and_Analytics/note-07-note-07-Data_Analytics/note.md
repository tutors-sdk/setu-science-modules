# Data Analytics

Data analytics supports critical decision making in business.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Database and Analytics

---

## Learning Outcomes

- Prepare data for analysis through cleansing and transformation.
- Prepare and construct data analytics on data sets.
- Prepare an appropriate visual representation of the data analytics findings.
- Evaluate the use and appropriateness of decision modelling.
- Evaluate the use of the data mining process for data analytics projects.
- Assess the use of measurement systems for monitoring performance.

---

## Indicative Content

- Business data requirements analysis
- Extraction, transformation and loading of cleaned data
- Modelling of data in business appropriate data storage environment
- Analytical skills for interpretation and presentation of data
- Data and decision analytics framework, value chain, tools and techniques
- Data mining: Big Data, XLMiner, Tableau
