# DB Design and Implementation

This module will introduce the student to the principles and practice of designing and implementing database systems.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Database and Analytics

---

## Learning Outcomes

- Appraise the role of a database and its management system.
- Construct Entity Relationship (ER) diagrams from business scenarios and reproduce those diagrams into normalised relations ready for database implementation.
- Construct a physical database design.
- Create a relational database using SQL Data Definition Language (DDL).
- Construct queries on a relational database using SQL Data Manipulation Language (DML).
- Assess the suitability and use of NoSQL databases.

---

## Indicative Content

- Database Environment
- Database Analysis and Design: Relational Modelling, Normalisation, NoSQL databases, CAP theorem
- Physical Database Design: Implementation descriptions of physical database
- Database Implementation
