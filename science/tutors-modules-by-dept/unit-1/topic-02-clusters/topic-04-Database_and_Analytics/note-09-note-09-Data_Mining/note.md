# Data Mining

The student will be introduced to the fundamental concepts and techniques of Data Mining.

---

## Module Details

- **Credits:** 10
- **Level:** Postgraduate
- **Department:** Computing and Mathematics
- **Cluster:** Database and Analytics

---

## Learning Outcomes

- Categorise between typical fundamental Data Mining problems.
- Appraise the concepts and fundamentals of Classification, Prediction and Clustering and their solution techniques.
- Assess the Data Mining Process and all of its steps.
- Justify the use of appropriate tools and techniques for each stage of the Data Mining Process.
- Evaluate, interpret and utilize results obtained at each step of the Data Mining Process.
- Prepare an appropriate visual representation of the date mining findings.

---

## Indicative Content

- Introduction to Data Mining; Classification, Prediction, Clustering
- Introduction to Data Mining Process
- Pre-processing: data gathering, wrangling, and transformation
- Model building, optimization and evaluation
- Result analysis, validation, deployment
- Use of data mining tools
