# Emerging Media Technologies

The module explores new trends, tools and technologies in the broad digital media domain.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Media Production

---

## Learning Outcomes

- Generalise the interaction possibilities beyond mouse-and-pointer interfaces. (Understanding).
- Critique emerging media trends based upon a set of discovered criteria (Evaluating).
- Construct a system that demonstrates user action synchronization and data consistency using emerging media tools/technologies (Applying).

---

## Indicative Content

- Interaction styles and interaction techniques
- Representing information to users
- Approaches to design, implementation and evaluation of non-mouse interaction
