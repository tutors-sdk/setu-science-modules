# Content Production

Participants create high-quality media assets for simulated clients or events.

---

## Module Details

- **Credits:** 5
- **Level:** Intermediate
- **Department:** Computing and Mathematics
- **Cluster:** Media Production

---

## Learning Outcomes

- Assess several issues raised by collaborative software.
- Organise themselves within a team environment.
- Apply a variety of design methods to a given problem.
- Compose, adapt and/or develop assets to a set of specifications.
- Review their performance based on appropriate metrics.

---

## Indicative Content

- Group communication
- Regulatory environments and bodies
- Audiovisual media conception, creation and evaluation
- Arts informed computing

---

## Resources Required

- EQUIPMENT: MAC PCs
- COMPUTER LAB: Multimedia Lab
