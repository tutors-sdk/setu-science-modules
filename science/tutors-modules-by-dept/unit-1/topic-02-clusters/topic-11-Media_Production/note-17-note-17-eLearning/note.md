# eLearning

This module is concerned with the authoring of instructionally sound, accessible eLearning for different platforms.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Media Production

---

## Learning Outcomes

- Evaluate and critique the suitability of eLearning Authoring Tools.
- Design organised instruction and develop learning environments using Instructional Design Guidelines.
- Prepare needs and task analysis.
- Substantiate the creation, testing and validation of accessible user interfaces using guidelines and standards.
- Develop a computer based, accessible learning environment.

---

## Indicative Content

- Design and Authoring of eLearning
- Needs and Task Analysis
- Organising Instruction
- Learning Environments
- Using Standards and Guidelines
- Creating, Testing and Validation of Accessible Web Sites
