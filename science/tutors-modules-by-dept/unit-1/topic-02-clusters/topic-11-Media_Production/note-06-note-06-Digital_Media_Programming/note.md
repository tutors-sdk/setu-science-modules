# Digital Media Programming

This module builds on strengths already gained in programming, digital media, and user experience.

---

## Module Details

- **Credits:** 5
- **Level:** Intermediate
- **Department:** Computing and Mathematics
- **Cluster:** Media Production

---

## Learning Outcomes

- Differentiate User Interfaces across multiple platforms. (Remembering).
- Design user interface consistent with digital media programming concepts (Understanding).
- Build a simple application with a modern graphical user interface (Creating).
- Build an interactive application utilising digital media libraries and frameworks. (Creating).

---

## Indicative Content

- Interaction design patterns: visual hierarchy; navigational distance
- Choosing interaction styles and interaction techniques
- Presenting information: navigation; representation; manipulation
- Modern UI libraries and frameworks
- Social and digital media APIs & frameworks
- Programmatic drawing; geometry; animation
- Programmatic image processing
- Programmatic sound & video processing
- Cross platform development
- Optimization for resource-constrained devices

---

## Resources Required

- Room Type: Computer Lab
- COMPUTER LAB: Multimedia Lab
- EQUIPMENT: MAC PCs
