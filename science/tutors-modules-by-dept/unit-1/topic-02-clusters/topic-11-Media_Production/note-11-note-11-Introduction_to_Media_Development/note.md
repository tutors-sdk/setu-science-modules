# Intro. to Media Development

This module introduces students to the concepts, tools and technologies surrounding media development.

---

## Module Details

- **Credits:** 5
- **Level:** Introductory
- **Department:** Computing and Mathematics
- **Cluster:** Media Production

---

## Learning Outcomes

- Recall the theoretical and practical principles of user experience.
- Discuss at least one national or international user interface design standard.
- Employ pre-recorded audio/visual media to a prescribed set of requirements.
- Publish an asset online.

---

## Indicative Content

- User Experience fundamentals
- Media delivery platforms, media formats & media tools
- User-centric programming
- Lo-fidelity prototyping
- Digital image manipulation
- Timeline based editing for audiovisual media
- Social media: Conversation prism

---

## Resources Required

- COMPUTER LAB: Multimedia Lab
- EQUIPMENT: MAC PCs
