# Web Programming

This module introduces the student to the principles of web programming.

---

## Module Details

- **Credits:** 5
- **Level:** Advanced
- **Department:** Computing and Mathematics
- **Cluster:** Software and Web Development

---

## Learning Outcomes

- Develop a web front-end using a responsive framework.
- Construct and deploy a web application based on the client/server application model.
- Develop a web application that utilises client-side scripting techniques.
- Integrate database access in a web application using server side scripting techniques.

---

## Indicative Content

- Responsive web front-end frameworks
- Client-side scripting: interactivity; client side data processing
- Server-side programming: database connectivity; CRUD (Create, Read, Update, Delete) functions
- Publishing web pages using FTP (File Transfer Protocol)
- Website deployment
- Website security
