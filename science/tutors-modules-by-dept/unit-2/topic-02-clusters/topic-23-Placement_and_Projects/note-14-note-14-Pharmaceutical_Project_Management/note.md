# Pharmaceutical P M

This module aims to provide the learner with the ability to develop an understanding of project management and its application.

---

## Module Details

- **Credits:** 10
- **Level:** Advanced
- **Department:** Science
- **Cluster:** Placement and Projects

---

## Learning Outcomes

- Evaluate the application of the principles of project management.
- Design, build and troubleshoot project management plans.
- Estimate project durations and Create, control, monitor project schedule using appropriate software tools (Software and non software).
- Estimate project costs, report on earned value, analyse cost variances and produce estimates at completion.
- Evaluate the project group aspects of project management - organisation, roles, attributes of the project manager, the role of sponsor etc.

---

## Indicative Content

- Introduction to project management
- Project life cycle
- Project design and planning: work breakdown structure, critical path network, project risk management, project change management, project quality control
- Project execution: reporting, documentations required and communication etc.
- Project cost measurement and management: the concept of earned value
- Managing a project team: leadership, resource allocation, HR implications, the role of the project manager
- Using project management tools and techniques
