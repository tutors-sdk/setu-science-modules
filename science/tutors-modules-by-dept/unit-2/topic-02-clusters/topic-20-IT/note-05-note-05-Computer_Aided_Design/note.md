# Computer Aided Design

This module will enable students to produce simple landscape design plans using computer aided design (CAD), and to print out drawings to scale.

---

## Module Details

- **Credits:** 5
- **Level:** Intermediate
- **Department:** Science
- **Cluster:** IT

---

## Learning Outcomes

- Operate CAD software drawing and modifying tools.
- Complete simple landscape drawing using CAD.
- Create drawings and plot them.
- Produce printed drawings to scale.

---

## Indicative Content

- Identify components of a CAD system
- Use quick setup wizard
- Use menu commands
- Use snap, grid, ortho, object snap
- Use scale grids
- Use drawing tools
- Use layers
- Dimensions and leaders
- Produce a hard copy of a file
- Plot to scale
