# Object Oriented Programming

This module follows on from the module called Advanced Programming Concepts and builds on concepts introduced there.

---

## Module Details

- **Credits:** 5
- **Level:** Intermediate
- **Department:** Engineering Technology
- **Cluster:** Software and Web Development

---

## Learning Outcomes

- Describe approaches to programming and problem solving.
- Write, debug and test well-styled and appropriately-documented programs.
- Demonstrate the use of OO techniques, using knowledge of the concepts when writing code and when using class libraries.
- Develop applications to control sensor inputs and outputs connected to a microcontroller.
- Manipulate microcontroller class libraries to control sensors and display outputs on various display devices.

---

## Indicative Content

- Object Oriented Programming
- C++ Classes and Constructors
- Abstract Data Types (ADTs)
- Overloaded Operators
- Try Catch Blocks (Error Processing)
- Inheritance and Polymorphism
