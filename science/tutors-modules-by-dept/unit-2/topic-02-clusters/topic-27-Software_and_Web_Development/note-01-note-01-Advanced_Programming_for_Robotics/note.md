# Advanced Programming Robotics

Advanced Programming Concepts module teaches the learner further programming concepts and techniques including strings, recursion, pointers, structures and vectors.

---

## Module Details

- **Credits:** 5
- **Level:** Intermediate
- **Department:** Engineering Technology
- **Cluster:** Software and Web Development

---

## Learning Outcomes

- Apply problem solving strategies to various computing problems of increasing complexity.
- Analyse different programming concepts and apply a particular programming problem.
- Design, develop, test and document applications using advanced programming constructs and data structures.
- Develop applications to control sensor inputs and outputs connected to a microcontroller.

---

## Indicative Content

- Strings
- Pointers
- Recursion
- Structures
- Vectors
